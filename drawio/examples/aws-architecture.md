# draw.io Example — AWS Three-Tier Web App

A classic three-tier AWS architecture using the `mxgraph.aws4.*` shape library: CloudFront → ALB → ECS → RDS, with S3 for static assets.

## Why use AWS4 stencils

draw.io ships with the official AWS Architecture Icons (the "AWS4" set, ~1,000 shapes). Reference them by their `mxgraph.aws4.<service>` style instead of drawing rectangles — diagrams stay accurate as AWS rebrands and look professional out of the box.

## How to find more shape names

Open [app.diagrams.net](https://app.diagrams.net), enable the **AWS** shape library (More Shapes → Networking → AWS17/AWS19/AWS4), drag any service onto the canvas, then **right-click → Edit Style**. The popup shows the exact `shape=mxgraph.aws4.<name>` string to copy.
