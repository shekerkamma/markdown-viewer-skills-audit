Implementation Work Packages
WP1: Foundation Build
Epic environment setup, org structure mapping, user role definitions, master file build (departments, providers, locations)
Months 1–4 · Team: Epic implementation partner + IT
WP2: Data Conversion
Map Cerner CCL → Epic Chronicles. Patient demographics, allergies, problem lists, medications. Historical records via MediQuant archival.
Months 3–8 · Team: DBA (3) + Clinical informaticists (4)
WP3: Integration Rebuild
Replace 47 Mirth Connect HL7v2 interfaces with Epic Bridges. Lab (ORU), Pharmacy (RDE), Radiology (ORM), ADT feeds. FHIR APIs for new consumers.
Months 5–10 · Team: Integration engineers (2)
WP4: Staff Training
Role-based training for 3,200 clinical staff. Physicians (8 hrs), nurses (12 hrs), front desk (6 hrs). Super-user certification program.
Months 8–12 · Budget: $150K · Trainer: vendor-provided
WP5: Go-Live + At-Elbow Support
Phased rollout: 2 pilot hospitals → remaining 10. 24/7 command center, 2-week at-elbow support per site. Stabilization monitoring.
Months 10–14 · Team: Full project team + vendor (50+ FTE)
WP6: Cerner Decommission
Cerner read-only archival (BytePad). Contract termination. On-prem hardware decommission. Oracle RAC license sunset.
Months 14–18 · Savings: $2.4M/year in Cerner licensing
Implementation Gaps (ArchiMate Implementation_Gap)
Data Migration Fidelity
Risk: 15M patient records with inconsistent coding between Cerner and Epic data models. Lossy conversion of free-text clinical notes.
Mitigation: MediQuant validation tool + 3 mock conversions before go-live. 99.5% accuracy target.
Dual-System Operational Cost
Risk: Running both systems for 6+ months doubles infrastructure cost (~$1.8M). Clinical staff context-switching between systems.
Mitigation: Phased site cutover (not big-bang). Each site runs dual for max 4 weeks.
Revenue Cycle Disruption
Risk: Billing system transition causes 15-30% revenue dip in first month post go-live (industry benchmark).
Mitigation: 90-day cash reserve + dedicated rev cycle optimization team. VA system saw 22% dip.
